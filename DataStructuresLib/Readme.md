## Data Structures

The linear data structures are as follows:
- Array
- Linked List
- Stack
- Queue
- Skip List

The non-linear data structures are as follows:
- Binary Tree
	- Rooted Binary Tree
	- Full Binary Trees
	- Perfect Binary Tree
	- Complete Binary Tree
	- Balanced Binary Tree
	- Degenerate Binary Tree
- Tries
- Graph
- Heap
- Hash Table
- Set
- Map
- Union-Find
